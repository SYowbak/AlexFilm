# Kino App

Додаток для підбору фільмів за настроєм, пошуку та збереження улюблених фільмів.
Збудований на базі Vue 3, Vite, Tailwind CSS та TMDB API.

## Основні функції

- Інтелектуальний підбір фільмів за "настроєм".
- Розширений фільтр за жанрами, роками, рейтингом та критерієм сортування.
- Текстовий пошук за назвою фільму.
- Можливість дізнатися більше: опис, постери, акторський склад, трейлер на YouTube.
- Пошук дубляжу на українських платформах: UAkino, UASerials, UAFix.
- Списки "Улюблених" зберігаються локально (Local Storage).
- Можливість знайти "випадковий" якісний фільм.

## Технології

- **Frontend Framework:** Vue 3 (Composition API)
- **Збірка:** Vite
- **Роутинг:** Vue Router
- **State Management:** Pinia
- **Стилізація:** Tailwind CSS

## Запуск додатку

У кореневій теці додатку є зручний скрипт:
`run_app.bat` - двічі клікніть по ньому, і він автоматично підніме сервер та відкриє вкладку в браузері на `http://localhost:5173`.

### Розробка

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть сервер:
   ```bash
   npm run dev
   ```

### Білд для продакшену

```bash
npm run build
```
